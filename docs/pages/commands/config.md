`gsv config`
============

manage gsv config

* [`gsv config:init`](#gsv-configinit)

## `gsv config:init`

initialize a fresh .gsvrc file

```
USAGE
  $ gsv config:init

OPTIONS
  -d, --dry          don't write to .gsvrc
  -f, --force        overrides existing .gsvrc
  -h, --help         show CLI help
  -t, --title=title  (required) blog title
  -u, --url=url      (required) blog url
```
