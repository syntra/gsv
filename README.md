# 🏗 Usage

<!-- usage -->
```sh-session
$ npm install -g gsv
$ gsv COMMAND
running command...
$ gsv (-v|--version|version)
gsv/0.0.1 linux-x64 node-v10.10.0
$ gsv --help [COMMAND]
USAGE
  $ gsv COMMAND
...
```
